# Parkinson's Disease Detection Using Machine Learning and Voice Biomarkers
Project Overview

This project applies machine learning techniques to detect Parkinson's Disease using biomedical voice measurements. The objective is to classify voice recordings as either belonging to a healthy individual or a patient diagnosed with Parkinson's disease.

The project includes data preprocessing, exploratory data analysis (EDA), feature correlation analysis, model training, and performance evaluation using multiple machine learning algorithms.

Dataset
The dataset consists of biomedical voice measurements collected from healthy individuals and Parkinson's patients.
## Dataset Source

The dataset used in this project is the Parkinson's Disease Dataset available on Kaggle:

* [Parkinson's Disease Dataset (Kaggle)](https://www.kaggle.com/datasets/thecansin/parkinsons-data-set?utm_source=chatgpt.com)

This dataset contains biomedical voice measurements collected from healthy individuals and patients diagnosed with Parkinson's disease.
